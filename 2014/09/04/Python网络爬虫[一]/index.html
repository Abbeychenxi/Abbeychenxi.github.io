URI

URI通常由三部分组成：

  1. 访问资源的命名机制；
  2. 存放资源的主机名
  3. 资源自身的名称，由路径表示

URI定义:统一资源标识符
URL定义:统一资源定位符

爬虫最主要的处理对象就是URL，它根据URL地址取得所需要的文件内容，然后对它进行进一步的处理。
