    数据集成和变换

    • smoothing平滑: 去除数据噪音 分箱 聚类 回归
    • aggregation聚集: 构造数据方
    • 泛化: 用高层次概念替换低层次概念
    • 规范化: 将属性按比例缩放
    • 属性构建: 可以构造新的属性并添加到属性集中

    规范化

    • 最小-最大规范化 : 对原始数据进行线性变换 (value - minA)/(maxA-minA)*(newmaxA-newminA)+newminA
    • z-score 规范化 :属性A的值基于A的平均值和标准差规范化(value - 平均值)/标准差
    • 小数定标规范化: 通过移动A的小数点位置进行规范化。小数点的移动位数依赖于A的最大绝对值v/(10的j次方)

    ps:j是使得max(|v’|)<1的最小整数

    数据归约

    • 数据方聚集: 聚集操作用于数据方中的数据
    • 维数约: 检测并删除不相关、弱相关或冗余的属性或维
    • 数据压缩: 使用编码机制压缩数据集
    • 数值压缩: 用替代的、较小的数据表示替换或估计数据
    • 离散化和概念分层产生: 属性的原始值用区间值或较高层的概念替换

    数据方聚集

    • lowest level of a data cube 数据立方体的最低水平
      感兴趣的聚合数据为一个单独的实体
    • Multiple levels of aggregation in data cubes多层次的聚合数据立方体
    • Reference appropriate levels引用适当的水平
      使用足以解决的任务的最小表示
    • 查询有关聚合信息，在可能的情况下应该使用数据立方

    numerosity reduction 数据块消减

    • 参数化方法
      假设数据符合一些模型,估计模型参数,只存储参数,并丢弃数据(除了可能的异常值)
      regression 回归
      Log-Liner Models
    • 无参数化方法
      不要假设模型
      主要方法:直方图、集群、抽样

    Regression and Log-Linear Models

    • 线性回归 经常使用最小二乘法来配合
    • 多元回归 允许将一个响应变量Y建模为多维特征向量的线性函数
    • Log-linear model 近似离散多维概率分布

    回归分析

    • Linear regression: Y = α + β X
      α和β指定行和估计利用手头的数据
      对已知值Y1,Y2,X1,X2,…使用最小二乘准则
    • 多元回归Y = b0 + b1 X1 + b2 X2
    • Log-linear models
      多路表联合概率较低层次的产物来近似表。
      概率:p(a,b,c,d)=αabβacχadδbcd
